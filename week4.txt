pass by value คือ การส่งผ่านค่าข้อมูล parameter จากผู่เรียนไปยังฟังก์ชันที่ถูกเรียก
pass by reference คือ การส่งค่าข้อมูลจากผู้เรียก

gcc จะ link ฟังก์ชันใน Standard Library ใหเราเองโดยอัตโนมัติ
แต่ฟังก์ชัน math library จะไม่ได้ถูก link ในอัตโนมัติ โดยเราตองระบุ -lm เพิ่มตอนเรียกใช้ gcc หรือหากต้องการใช้ math library ต้อง #include <math.h> เข้าไป

ฟังก์ชันที่สามารถเรียกใช้ตัวเองได้ เรียกว่า recursion
