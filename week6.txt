Int A[10];
A คือชื่อของarray
ตัวแปรต่อๆไปในarrayใช้วิธีนับจากจุดเริ่มต้น เช่น A[10] ก็นับไป10ตัวจากตำแหน่งเริ่มต้น

#default จะเปลี่ยนอักษรทั้งหมดในโปรแกรม เหมือนเขียนใหม่
const int เป็นการประกาษตัวแปร ไม่มีการเปลี่ยนแปลงค่าตลอดการทำงาน

pass by value ส่งค่าไปยังงฟังก์ชัน ทำงานได้เลย
pass by reference ส่งตำแหน่งไปยังฟังก์ชัน
